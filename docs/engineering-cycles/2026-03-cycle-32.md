## Risks / Trade-offs

- Payload labels differ between systems and require stat-key normalization
  before stable aggregate comparison.
- The comparison UI can become dense quickly; phased rollout is required to
  avoid regressions in readability and performance.
- Some score formulas (especially composite PVE/PVP ranking) are design
  decisions, not direct API fields, and need validation with gameplay intent.
